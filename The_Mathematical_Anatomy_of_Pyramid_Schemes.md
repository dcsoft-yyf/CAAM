## VI. Core Members Breaking Away: Self-Replication of Locked Systems

Another typical phenomenon in pyramid schemes is "core members going independent"—superficially "betrayal" or "entrepreneurship," but actually **self-replication and upgrading of locked systems**.

A member capable of "going independent" has typically completed five-dimensional lock-in within the original organization: G→1, L→L_max, P→1, Q→0, S→1. They cannot "exit" the pyramid scheme, only "upgrade" it.

### 6.2 Driving Force for Breaking Away: Blocked G Expansion

The original organization may impose limits on G "expansion": upline suppression, promotion ceilings, rule constraints. When the original system cannot satisfy the core member's G expansion needs, the locked state drives them to **seek new outlets**—not exit, but **creating their own system**.

### 6.3 Five-Dimensional Upgrade After Breaking Away

| **Q** | Q→0 | Q maintained at 0, but object of quitting changed |
| **S** | S→1 (bound to original organization) | **S transferred**: Bound to their new organization |

### 6.4 Lock-In Design of the New System

Core members will repair "loopholes" in the original organization—eliminating upline cuts (becoming the sole upline), shortening promotion paths, increasing personal authority. These "optimizations" make the new system's lock-in efficiency even higher.

| **P（感知）** | 将失败视为止损 | 将拒绝重编码为“考验” | “反对者是格局小” | 亲友劝阻反被拉黑 |
| **Q（退出）** | 保留退出通道 | 退出=承认自己蠢、背叛 | “退出就是失败” | 受害者“越陷越深” |
| **S（绑定）** | 多重社会身份 | 唯一身份：传销成员 | 统一住宿，切断外部联系 | “艾兴合”App社群绑定 |


### 2.2 锁死进程的数学临界点
根据碳基AI分析法，当五个维度中有三个同时失效（G→1、L→L_max、P→1、Q→0、S→1中任意三个），系统即越过临界点，进入不可逆锁死。

在传销中，这个临界点通常发生在：

+ **G坍缩**：认定“这是唯一机会”
+ **L透支**：积蓄耗尽、借遍亲友
+ **P扭曲**：将一切负面反馈重编码为“考验”

一旦跨过这个临界点，参与者已经无法靠自身力量退出。他必须“成功”——否则，他将面对一个无法承受的自我。2025年成都“1040工程”案中，200余名受害者大多在完成这三步后，再也无法自拔。

## 三、双环锁死：上下层咬合结构
传销的锁死不是单点故障，而是**上下层双环锁死**。每一方都困在自己的牢笼里，又互相递钥匙——但钥匙都在对方手里。**这是两个溺水者抱在一起下沉的典型结构。**

### 3.1 上层（组织者/早期参与者）循环
| 维度 | 状态 |
| :--- | :--- |
| **G** | 从“赚钱”坍缩为“必须证明这个模式是对的” |
| **L** | 已投入巨大，可能涉及法律风险，无法回头 |
| **P** | 将下线破产重编码为“他们不够努力” |
| **Q** | 退出等于承认自己是骗子，面临法律后果 |
| **S** | 已完全绑定，无其他职业能力 |


**循环**：发展下线→获得收益→道德快感→依赖下线→恐惧崩盘→更激进发展

### 3.2 下层（被诱导参与者）循环
| 维度 | 状态 |
| :--- | :--- |
| **G** | 从“赚钱”坍缩为“必须回本” |
| **L** | 已投入全部积蓄、借贷、亲友借款 |
| **P** | 将亲友拒绝重编码为“考验”，将亏损重编码为“即将成功” |
| **Q** | 退出等于承认自己蠢、对不起上线 |
| **S** | 社交圈只剩传销成员，与亲友决裂 |


**循环**：投入→亏损→需要更多下线回本→发展失败→更绝望→更激进投入

### 3.3 咬合点：为何分不开
上下层看似对立，实则构成了完美的供需咬合：

+ **上层需要下层**：下层是收益来源，也是“自己没做错”的心理证明
+ **下层需要上层**：上层是“成功希望”的唯一来源
+ **资源单向流动**：必须从下层向上层，反向意味着系统崩溃
+ **认知相互印证**：彼此为对方提供幻觉
+ **退出相互锁死**：任何一方退出都会导致另一方崩溃

## 四、三环锁死：同级循环增强
传销的锁死不只存在于上下层之间，**同级成员之间的循环增强**构成三环锁死，使整个系统更加坚固。

### 4.1 同级耦合的五维机制
| 维度 | 同级耦合机制 | 具体表现 |
| :--- | :--- | :--- |
| **G** | 目标共振 | “大家都还在坚持，说明这条路是对的” |
| **L** | 资源竞赛 | “他都借了20万，我才借5万，不够拼” |
| **P** | 感知同步 | 共同将亲友反对重编码为“格局小” |
| **Q** | 退出焊死 | “你走了我们怎么办？”集体羞耻感 |
| **S** | 身份替代 | 社交圈只剩传销成员 |


### 4.2 群体共振的数学本质
设同级成员的数量为n，成员i的状态向量为X_i = (G_i, L_i, P_i, Q_i, S_i)。同级耦合的动力学可近似为：

dX_i/dt = f(X_i) + κ * Σ_{j≠i} (X_j - X_i)

其中κ是耦合强度。*_这个公式说明：成员之间的相互影响强度κ越高，大家的五维状态就越会同步锁死。__当κ超过临界值时，所有成员同步收敛到锁死吸引子 _*。这就是为什么传销组织喜欢“集体住宿”——物理接近极大提升了κ，使锁死更加迅速彻底。

### 4.3 退出者的“社会性死亡”
当一个成员试图退出时，同级成员的反应进一步焊死退出通道：

| 反应 | 效果 |
| :--- | :--- |
| 集体劝说 | 心理压力，Q被群体拉回 |
| 道德绑架 | “你对得起大家吗？”Q→0 |
| 威胁孤立 | “你走了我们再也不理你”S崩溃恐惧 |
| 经济追索 | “你欠大家的钱怎么办？”L反噬 |


**退出不再是个体决策，而是与整个群体的博弈。**



## 五、被利用感消除：锁死工程的核心成就
传销最精妙的设计不是强迫，而是**让成员感受不到被利用**。如果成员感受到被利用：

+ G无法锁定——“我为别人赚钱，凭什么拼命？”
+ Q无法焊死——“我凭什么要为一个利用我的系统牺牲？”
+ S无法固化——“谁愿意绑定一个利用自己的系统？”

因此，传销必须消除被利用感。这是一套系统性的五维工程：

### 5.1 G重构：从“打工者”到“创业者”
| 话术/操作 | 认知重构效果 |
| :--- | :--- |
| “你不是在为我干，是在为自己干” | 将“被雇佣”重编码为“自主创业” |
| “这是你的生意，我只是帮你” | 将上下级关系重编码为“合作关系” |
| 要求成员自称“老总”“创始人” | 身份认同塑造G |


### 5.2 P重编码：将“剥削”转化为“帮助”
| 客观现实 | 重编码后 |
| :--- | :--- |
| 上级拿走大部分收益 | “上级拿的是他应得的，因为他帮我” |
| 下级承担全部风险 | “风险是投资，上级承担的是培训成本” |
| 收益分配不公平 | “这不是剥削，是激励——你以后也能这样” |


### 5.3 S重新绑定：从“被绑定”到“主动绑定”
| 被动绑定（被利用感强） | 主动绑定（被利用感弱） |
| :--- | :--- |
| “我被困在这里” | “我选择留在这里” |
| “系统控制我” | “我拥有系统” |


**实现机制**：让成员参与“决策”（团队名称、口号）、发展自己的下线、感受“掌控感”（虽然是幻觉）。

### 5.4 层级幻觉：创造“未来剥削者”的想象
让下层成员想象自己未来会成为上层：

| 当下现实 | 未来想象 | 效果 |
| :--- | :--- | :--- |
| 我被上层剥削 | 我以后也会剥削下层 | 剥削被正常化 |
| 我是资源输出者 | 我以后会是资源接收者 | 当下付出被重编码为“投资” |


### 5.5 竞争共同体：将内部剥削转化为外部竞争
将内部矛盾转化为“共同对抗外部敌人”：

| 内部剥削（真实） | 外部敌人（建构） | 效果 |
| :--- | :--- | :--- |
| 上下级利益冲突 | “共同对抗反对我们的人” | 内部矛盾被外部化 |
| 同级竞争资源 | “共同对抗传统观念” | 竞争被重编码为“团结” |


**当成员将精力用于对抗外部敌人时，内部的被利用感被系统性遮蔽。**



## 六、骨干单干：锁死系统的自我复制
传销的另一个典型现象是“骨干单干”——表面上是“背叛”或“创业”，实际上是**锁死系统的自我复制与升级**。

### 6.1 骨干单干前的五维状态
一个能“单干”的骨干，通常已在原组织中完成了五维锁死：G→1、L→L_max、P→1、Q→0、S→1。他无法“退出”传销，只能“升级”传销。

### 6.2 单干的驱动力：G膨胀受阻
原组织可能对G的“膨胀”存在限制：上级压制、晋升天花板、规则束缚。当原系统无法满足骨干的G膨胀需求时，锁死状态驱使他**寻找新的出口**——不是退出，而是**自己创建一个系统**。

### 6.3 单干后的五维升级
| 维度 | 迁移前 | 迁移后 |
| :--- | :--- | :--- |
| **G** | G→1（必须成功） | **G升级**：还要“证明比原组织更强” |
| **L** | L→L_max | L保持透支，但杠杆对象转移 |
| **P** | P→1 | P强化：将“背叛”重编码为“觉醒” |
| **Q** | Q→0 | Q维持0，但退出对象变了 |
| **S** | S→1（绑定原组织） | **S转移**：绑定自己的新组织 |


### 6.4 新体系的锁死设计
骨干会修复原组织的“漏洞”——取消上级抽成（自己成为唯一上级），缩短晋升路径，增加个人权威。这些“优化”使新体系的锁死效率更高。



## 七、生态锁死：传销组织之间的相互锁定
传销并非孤立的组织。多个传销组织之间形成了**生态锁死**。

### 7.1 组织间五维耦合
| 维度 | 组织间耦合机制 |
| :--- | :--- |
| **G** | 竞争锁定：“必须打败竞争对手” |
| **L** | 资源博弈：相互挖人、军备竞赛 |
| **P** | 相互妖魔化：“我们是正规，他们是骗局” |
| **Q** | 相互焊死：退出=向对手认输 |
| **S** | 边界固化：“我们是A的人，不是B的人” |


### 7.2 单个组织打击的悖论
当执法部门打击其中一个组织时：

| 其他组织的反应 | 对锁死状态的影响 |
| :--- | :--- |
| “A被消灭了，说明我们是对的” | P扭曲加深，G锁定强化 |
| “A的地盘空出来了，快去占领” | L投入增加 |
| “A的成员无家可归，来加入我们” | 人员流动，S重新绑定 |


**关键悖论**：单个组织的覆灭，反而被其他组织解读为“自己正确的证明”。整个生态的锁死状态不但没有减弱，反而可能强化。这解释了为什么传销屡禁不止——**执法行动被锁死生态系统“吸收”了**。2025年，当某地打击一个传销组织后，周边多个同类组织短期内会员数激增，正是这一机制的体现。

### 7.3 生态锁死的自我强化
+ **相互印证**：每个组织的存在都证明其他组织的“竞争对手”身份
+ **相互驱动**：竞争使所有组织的G、L同时升高
+ **相互焊死**：任何组织退出都会成为其他组织的“胜利”
+ **相互吸收**：一个组织的瓦解，资源被其他组织瓜分

## 十、结论：传销是锁死动力学的完整标本

1. 传销通过五维工程，将个体推向不可逆锁死
